# delivery-app-backend
BackEnd do projeto DeliveryAPP


## Dynaconf
### Ambiente default
```
export FLASK_ENV=default
```
### Ambiente desenvolvimento
```
export FLASK_ENV=development
```
### Ambiente teste
```
export FLASK_ENV=testing
```
### Ambiente produção
```
export FLASK_ENV=production
```
## Makefile
### Rodar aplicação local
```
make up
```
